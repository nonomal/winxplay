# winxplay
专为Windows设计的数字标牌播放器

支持(视频、音频、流媒体、图片、摄像头、动画、文本、滚动字幕、二维码)

### 安装方法

```
git clone https://github.com/nulijiabei/winxplay.git 或 https://gitee.com/nljb/winxplay.git
```
>进入 winxplay 目录，直接运行 winxplay.exe 即可
